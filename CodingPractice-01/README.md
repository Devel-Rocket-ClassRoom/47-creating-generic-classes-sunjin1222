# 코딩 연습 - 제네릭 클래스 만들기 (1)

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. 간단한 제네릭 클래스

하나의 값을 담는 제네릭 `Cup<T>` 클래스를 정의하고, `Cup<string>`과 `Cup<int>`를 생성하여 출력하세요.

**실행 결과**

```
음료: 커피
용량: 500ml
```

---

## 2. 필드, 속성, 메서드에서 타입 매개변수 사용

`Container<T>` 클래스를 정의하세요. 기본 생성자에서 `default(T)`로 초기화하고, 매개변수 있는 생성자도 제공합니다. `Display()` 메서드로 저장된 값을 출력하세요.

- `Container<string>("Hello")`, `Container<double>(3.14)`, `Container<bool>()`(기본값)을 생성하고 `Display()` 호출

**실행 결과**

```
저장된 값: Hello
저장된 값: 3.14
저장된 값: False
```

---

## 3. 두 개 이상의 타입 매개변수

두 개의 타입 매개변수를 가진 `Pair<T, V>` 클래스를 정의하고, `Display()` 메서드로 두 값을 출력하세요.

- `Pair<string, int>("홍길동", 25)`, `Pair<double, bool>(85.5, true)`, `Pair<int, int>(100, 200)` 테스트

**실행 결과**

```
First: 홍길동, Second: 25
First: 85.5, Second: True
First: 100, Second: 200
```

---

## 4. 키-값 쌍 클래스

`KeyValue<TKey, TValue>` 클래스를 정의하고, `ToString()`을 오버라이드하여 `"[Key] = Value"` 형식으로 출력하세요.

**실행 결과**

```
[점수] = 100
[1] = 첫 번째
```
