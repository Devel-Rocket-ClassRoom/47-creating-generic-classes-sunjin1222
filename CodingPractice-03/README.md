# 코딩 연습 - 제네릭 클래스 만들기 (3)

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

## 1. 타입 매개변수를 열어둔 상속

`Container<T>` 기반 클래스를 정의하고, `Store()`와 `Get()` 메서드에 로깅을 추가한 `LoggingContainer<T>`를 상속으로 구현하세요.

**실행 결과**

```
[로그] 저장: 테스트 데이터
[로그] 조회: 테스트 데이터
```

---

## 2. 닫힌 타입으로 상속

제네릭 `Stack<T>` 기반 클래스를 정의하고, `int` 타입으로 고정된 `IntStack`을 상속하여 `Sum()` 메서드를 추가하세요.

**실행 결과**

```
합계: 60
Pop: 30
남은 개수: 2
```

---

## 3. 정적 데이터와 제네릭

생성자에서 `Count`를 증가시키는 `Counter<T>` 클래스를 정의하고, `Counter<int>`, `Counter<string>`, `Counter<bool>`의 Count가 각각 독립적임을 확인하세요.

**실행 결과**

```
Counter<int> 생성 횟수: 2
Counter<string> 생성 횟수: 3
Counter<bool> 생성 횟수: 1
```

---

## 4. default 키워드

`DefaultDemo<T>` 클래스를 정의하고, `ShowDefault()` 메서드에서 `typeof(T).Name`과 `default` 키워드를 사용하여 타입별 기본값을 출력하세요.

**실행 결과**

```
타입 Int32의 기본값: 0
타입 String의 기본값:
타입 Boolean의 기본값: False
타입 Double의 기본값: 0
```

---

## 5. 종합 예제 - 제네릭 스택

배열 기반의 `MyStack<T>` 클래스를 구현하세요.

- 초기 용량 4, 용량이 부족하면 2배로 확장
- `Push(T item)`, `Pop()`, `Peek()` 메서드
- `Count`, `IsEmpty` 속성
- 용량 확장 시 `"[내부] 용량 확장: N"` 메시지 출력
- 문자열 5개를 Push하고 (용량 확장 발생), 모두 Pop하여 출력

**실행 결과**

```
[내부] 용량 확장: 8
개수: 5
Peek: 다섯 번째
Pop: 다섯 번째
Pop: 네 번째
Pop: 세 번째
Pop: 두 번째
Pop: 첫 번째
```
