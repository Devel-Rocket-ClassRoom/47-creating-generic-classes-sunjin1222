# 코딩 연습 - 제네릭 클래스 만들기 (2)

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

---

## 1. 제네릭 인터페이스 구현

`IStorage<T>` 인터페이스를 정의하고 `SimpleStorage<T>` 클래스로 구현하세요.

- `Store(T item)`: 아이템 저장 후 메시지 출력
- `Retrieve()`: 저장된 아이템 반환
- `string`과 `int` 타입으로 테스트

**실행 결과**

```
저장 완료: 중요한 데이터
꺼낸 값: 중요한 데이터
저장 완료: 12345
꺼낸 값: 12345
```

---

## 2. struct 제약 조건

`where T : struct` 제약 조건을 사용하는 `ValueContainer<T>` 클래스를 정의하세요. `IsDefault()` 메서드로 기본값 여부를 확인합니다.

**실행 결과**

```
값: 100, 기본값 여부: False
값: 0, 기본값 여부: True
```

---

## 3. class 제약 조건

`where T : class` 제약 조건을 사용하는 `ReferenceContainer<T>` 클래스를 정의하세요. `IsNull()` 메서드로 null 여부를 확인합니다.

**실행 결과**

```
null 여부: True
null 여부: False
```

---

## 4. new() 제약 조건

`where T : new()` 제약 조건을 사용하는 `Factory<T>` 클래스를 정의하세요. `Create()`와 `CreateArray(int count)` 메서드를 구현합니다. `Product` 클래스를 정의하고 테스트하세요.

**실행 결과**

```
생성된 상품: 새 상품
배열 크기: 3
```

---

## 5. 인터페이스 제약 조건

`where T : IComparable<T>` 제약 조건을 사용하는 `Sorter<T>` 클래스를 정의하세요. `GetMax()`와 `GetMin()` 메서드를 구현합니다.

**실행 결과**

```
최댓값: 20
최솟값: 10
최댓값: banana
최솟값: apple
```

---

## 6. 기반 클래스 제약 조건

`Entity` 기반 클래스와 `Player`, `Monster` 파생 클래스를 정의하세요. `where T : Entity` 제약 조건을 사용하는 `EntityManager<T>` 클래스를 정의하고, 플레이어를 추가한 후 ID로 검색하세요.

**실행 결과**

```
엔티티 추가됨 (ID: 1)
엔티티 추가됨 (ID: 2)
찾은 플레이어: 홍길동
```
